## P-01: 田中さくら（児童）

**「作文の宿題、何を書けばいいかわからない」**

| 項目 | 内容 |
|---|---|
| 年齢 | 10歳（小学5年生） |
| 学年範囲 | 代表ペルソナ（本プロダクトは小1〜高3を対象とするが、MVPでは全学年を1ペルソナで統一） |
| 主要デバイス | iPad（タブレット）横向き |
| サブデバイス | スマートフォン（縦向き） |
| 利用場所 | 自宅の勉強机・リビング |
| 利用時間帯 | 放課後〜夕食前（16:00〜18:00）、夏休み中（午前中） |

### ゴール

- 夏休みの読書感想文や行事の作文を「早く・うまく」仕上げたい
- 原稿用紙をきれいに埋めて、先生に褒められたい
- 「何を書けばいいか」がわかれば、自分で書ける

### ペインポイント

- 書き始めが特に難しい（「最初の1文が書けない」）
- 字数が足りなくなる（何を足していいかわからない）
- 原稿用紙のルールがわからない（句読点の位置・段落の字下げ等）
- 途中で保存するのを忘れてデータが消えた経験がある

### モチベーション

- 書き終わったときの達成感が好き
- キャラクターやアニメーション演出が楽しいと続けられる
- 「自分で全部書けた」という自己効力感が大事

### 技術リテラシー

- タブレット操作は慣れている（ゲーム・YouTube等）
- キーボード入力より音声・フリック入力が得意
- 縦書きアプリの使用経験はほぼなし

### 関連ユーザーストーリー

US-001〜US-013（全ストーリー）

## P-02: 田中まお（保護者）

**「子どもが書いた作文を印刷して提出したい」**

| 項目 | 内容 |
|---|---|
| 年齢 | 38歳 |
| 主要デバイス | MacBook / Windows PC |
| サブデバイス | iPhone（確認用） |
| 利用場所 | 自宅・職場 |
| 利用時間帯 | 夜（21:00〜23:00）、提出前日 |

### ゴール

- 子どもが書いた作文を、学校提出可能な形式で印刷したい
- 子どもが自分で作文を完成できるようになってほしい（自立支援）
- 子どもの成長を記録として残したい（Phase 2: 作文アルバム）

### ペインポイント

- 印刷時に原稿用紙のマス目がズレる / A4に収まらない
- 子どもが作文を嫌がって取りかかりに時間がかかる
- 無料ツールは広告が多く、子どもに見せにくい

### モチベーション

- 子どもが自主的に取り組んでくれれば手間が省ける
- 印刷がワンクリックで完了するシンプルさを求めている
- 信頼できる・安全なサービスを選びたい（子ども向けのため）

### 技術リテラシー

- PC・スマホ操作は問題なし
- プリンター使用経験あり（年数回）
- アプリの支払いはApple Pay / クレカで対応可

### 関連ユーザーストーリー

US-009（枚数設定）、US-012（印刷機能）
